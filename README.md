## Installation

```bash
$ git clone https://github.com/indrajuni96/RESTful_API_CAFEKU.git
$ cd [folder]
$ npm install
```

## Create Environment Variables

```bash
$ mv .env.example .env
$ [your text editor] .env
```

## Features

- Register User
- Login User With JWT
- CRUD Category
- CRUD Product
